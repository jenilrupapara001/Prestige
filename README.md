# Aurora - Premium Shopify Theme

Aurora is a premium, AI-optimized Shopify Online Store 2.0 theme designed specifically for modern fashion and clothing brands. Built with a conversion-focused architecture and editorial-grade design system, Aurora enables merchants to create high-impact shopping experiences without custom development.

- **Editorial-Grade Design System**: Premium, minimalist aesthetic perfect for fashion brands
- **Mobile-Optimized UX**: Responsive design that works flawlessly across all devices
- **Conversion-Focused Architecture**: Built to maximize engagement and sales
- **Accessibility Compliant**: WCAG 2.1 AA compliant for inclusive shopping experiences

### Merchandising Tools
- **Advanced Product Display**: Multiple layout options including grid, carousel, and shop-the-look

## Installation

1. **Download the theme** files to your local machine
2. **Compress the theme** folder into a `.zip` file
3. **Log in to your Shopify admin** panel
4. Navigate to **Online Store > Themes**
5. Click **Upload theme** and select your `.zip` file
6. Once uploaded, click **Customize** to start editing

## Quick Start

After installation:

1. **Set up your logo**: Go to Theme Settings > Header
2. **Configure colors**: Go to Theme Settings > Colors
3. **Choose typography**: Go to Theme Settings > Typography
4. **Add sections**: Use the theme editor to add and arrange sections on your homepage
5. **Configure navigation**: Set up your menus in Navigation settings
6. **Add products**: Import or create your products
7. **Customize product pages**: Configure product template settings

## Customization Guide

### Color Swatches

To enable color swatches for product variants:
1. Create a product with color variants
2. Add a color value to each variant (e.g., `#FF0000` for red)
3. Enable "Show color swatch" in the section settings

### Mega Menu

To create a mega menu:
1. Create a menu with nested items
2. In Header settings, enter the menu item name in the "Mega menu" field
3. The dropdown will automatically display in a multi-column layout

### Product Tabs

Add custom tabs to product pages:
1. Create a page with your tab content
2. In Product Template settings, select the page for Tab 1 or Tab 2
3. The tab will appear on all product pages

### Shop the Look

Create shoppable lifestyle images:
1. Add the "Shop the Look" section to your page
2. Upload a lifestyle image
3. Select products and position the dots on the image
4. Customers can click dots to see product details
